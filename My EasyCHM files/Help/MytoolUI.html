MytoolUI 4.0.0

 

这是一个工具集合，可以辅助完成门诊日志格式化、出院随访格式化、流行病学调查表的填写及打印、血气分析的校验及智能提示等功能，旨在提升办公效率，减少一些繁琐、重复的工序。项目基于.Net平台开发，依赖于.NetFramework4.7框架，在使用软件前应保证PC已正确安装.NetFramework4.7及其以上的版本。

程序使用了Aspose.Words第三方库作为word文件处理程序，Aspose.Words是一款先进的类库，使您可以直接在各个应用程序中执行各种文档处理任务。Aspose.Words支持DOC，OOXML，RTF，HTML，OpenDocument, PDF, XPS, EPUB和其他格式。使用Aspose.Words，您可以生成、更改、转换、渲染和打印文档而不依赖于Microsoft Word，值得注意的是，这是一个商用软件，对日常使用不存在问题，但请不要将本软件用于商业目的。

Aspose.Words for .NET is an advanced document processing API to perform a wide range of management and manipulation tasks in a variety of file formats. API supports to create, modify, convert, render and print files without using Microsoft Word or other third-party software directly in cross-platform applications.

 

程序整合了下列功能：

1.    病案工具

这是一个工具合集，包括感染病例报告卡打印、病历质量评分表打印、流行病学调查表及病房管理告知书的打印功能。

1.1感染病例报告卡

该功能在出院病案整理时提供帮助。可通过中联BH系统获取信息后，填入感染病例报告卡中，并打印到文件。

1.2病案质量评分表

该功能在出院病案整理时提供帮助。可通过中联BH系统获取信息后，填入病案质量评分表中，并打印到文件。

1.3流行病学调查表

针对入院患者，实现流行病学调查表的生成及辅助填写功能。其原理为从中联系统首页中获取患者的各项信息，利用所获取的信息生成流行病学调查表并填入对应的空位，对于首页信息不完善的患者该功能部分受限。

2.    出院随访

该功能提供出院随访格式化。其原理是通过读取从中联系统导出的出入院登记表，筛选出所选用户名所管理的患者信息，并参照医院出院随访登记本的格式，自动生成出院随访登记表。在使用该功能前，应该确保所导出的患者均已完善病案首页的填写。

3.    慢病报卡

4.    该工具可辅助筛选出需要填慢病卡片的患者信息并生成卡片。其类别主要包括“慢性肺病”、“急性心血管事件”、“急性脑血管事件”，对于肿瘤保卡，因填写内容过多，故单独提供页面。可以通过点击左侧“慢病保卡”按钮后，在其下方即会显示肿瘤报卡入口。

5.    门诊日志

操作方法同其他，该工具对门诊日志提供格式化、缺失项目的检测及智能补全。

6.    肿瘤报卡

该工具类属慢病保卡，因项目较多，故独占一个界面

7.    血气分析

这是一个简单的血气分析工具，旨在对程序的运行原理感兴趣的朋友进行学习交流，通过录入样本的酸碱值、氧分压、二氧化碳分压等项目，对值进行平衡校验，并提示可能的酸碱失衡结果，因工具并非人脑，无法综合判断其原发失衡为急性还是慢性，程序仅选取了内科最常见到的慢性酸碱失衡作为计算依据，故而结果可能有误，对其所提示的结果仅作参考用，如您对该程序的运行原理已十分了解，请及时删除，该项目已在GitHub开源。
